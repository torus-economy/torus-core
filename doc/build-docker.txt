Build torusd image:
$ DOCKER_BUILDKIT=1 docker build --target torusd-unix --tag torusd .

Build torus-qt for unix image:
$ DOCKER_BUILDKIT=1 docker build --target torus-qt-unix --tag torus-qt .

Run torusd image:
$ docker run -p 12411:12411 -p 22411:22411 -v /home/$USER/.TORUS:/root/.TORUS torusd

Run torus-qt for unix image:
$ docker run --network host -v /home/$USER/.TORUS:/root/.TORUS -v /home/$USER/.Xauthority:/root/.Xauthority:rw -e DISPLAY=$DISPLAY torus-qt
